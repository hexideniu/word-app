# 单词本

基于 AI 的英语单词学习工具，通过生成文章语境帮助记忆单词。

## 功能

- **查词** — 输入英文单词，调用大模型 API 获取音标、释义、例句
- **单词本** — 收藏单词，支持搜索、删除、重置复习进度
- **生成文章** — 随机选取待复习单词，生成三种风格的英文短文：
  - 深刻故事
  - 有趣对话
  - 新闻资讯
- **复习追踪** — 每个单词最多复习 7 次，达到后标记为「已掌握」
- **本次复习** — 查看本次选中的单词列表，支持标记「不认识」重置进度或删除单词
- **保存文章** — 生成的文章可保存或复制，方便回顾
- **发音朗读** — 点击喇叭图标朗读单词发音（Web Speech API）
- **多主题** — 6 种主题风格：经典蓝、暗夜模式、薄荷绿、薰衣草、暖阳橙、樱花粉
- **导入/导出** — 支持 JSON 格式导入导出单词本和设置

## 使用

直接用浏览器打开 `word.html` 即可使用。

首次使用需在设置中配置：
1. API 地址（兼容 OpenAI 格式）
2. API Key
3. 模型名称

## 设置项

| 设置 | 说明 | 默认值 |
|------|------|--------|
| API 地址 | 兼容 OpenAI 的 API 端点 | — |
| API Key | API 密钥 | — |
| 模型 | 模型名称 | gpt-4o-mini |
| 复习单词数量 | 每次生成文章随机选取的单词数 | 10 |
| 文章字数 | 生成文章的目标英文词数 | 100 |
| 语言难度 | 初级 / 中级 / 高级 | 中级 |

## 技术

- 纯前端，无框架依赖
- 单 HTML 文件，移动端优先
- localStorage 持久化数据
- OpenAI 兼容 API
